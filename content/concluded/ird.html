Internet Resource Discovery (IRD) Research Group Charter

Chairs: Cecilia Preston, cecilia@well.com
              Clifford Lynch cliff@cni.org
Interest Mailing List: irtf-rd@cni.org
To subscribe to the Interest list: irtf-rd-request@cni.org

Charter is currently being revised!

This IRTF research group is chartered to develop a model for resource description on the Internet, including

  1. the creation of surrogates for electronic resources for indexing, query routing, and retrieval
  2. mechanisms for the creation, maintenance, and discovery of data element sets for the surrogates
  3. technical requirements for information system components using these surrogates

As results appear, this group will be responsible for recommending where further work on each result might be done in the IETF or the IRTF.

Working group membership is by invitation only, at the sole discretion of the chair(s). Timelines and milestones will be developed by the working group.
